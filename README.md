# PhotoCV

![Logo](/assets/logo150.png)

Projeto apenas para estudo das bibliotecas PySide6 e OpenCv, feito para facilitar o uso de filtros e funcionalidades do OpenCv nas fotos com o uso de um aplicativo Desktop.
